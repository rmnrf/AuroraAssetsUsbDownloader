# Aurora Assets USB Downloader

Консольное приложение для скачивания обложек игр Xbox 360 и подготовки их для Aurora Dashboard — **без FTP, без интернета на Xbox, только ПК + флешка**.

## Как работает

Приложение использует исходный код [AuroraAssetEditor](https://github.com/XboxUnity/AuroraAssetEditor), но вместо FTP-загрузки на Xbox записывает `.asset` файлы **локально на флешку**.

```
┌────────────┐     ┌──────────────┐     ┌────────────┐
│   Xbox 360 │ ──→ │   USB Flash  │ ──→ │     ПК     │
│  Content.db│     │              │     │  Скачивание│
└────────────┘     │              │     │  обложек   │
                   │              │ ←── │  с Internet│
┌────────────┐     │  GameData/   │     │  Archive   │
│   Xbox 360 │ ←── │  *.asset     │     └────────────┘
│  Aurora ✓  │     └──────────────┘
└────────────┘
```

## Требования

- Windows 10/11
- .NET 10 Runtime
- **AuroraAsset.dll** из [AuroraAssetEditor](https://github.com/XboxUnity/AuroraAssetEditor/releases) (скопировать рядом с .exe)
  - Без этой DLL обложки сохранятся как PNG (не .asset)

## Использование

### Шаг 1: Скопировать Content.db с Xbox

Скопируйте файл `Content.db` с Xbox на флешку:
```
Xbox HDD: Aurora\Data\DataBases\Content.db → Флешка
```

### Шаг 2: Запустить приложение

```bash
# Из командной строки просто передайте букву флешки (например E):
AuroraAssetsUsbDownloader.exe E

# Или запустите .exe без параметров — он спросит букву диска интерактивно
AuroraAssetsUsbDownloader.exe
```

При запуске программа предложит **выбрать язык (English / Русский)**, а затем появится **интерактивное меню**, где с помощью стрелок и пробела можно выбрать, какие именно типы ресурсов скачивать (Boxart, Background, Icon/Banner, Screenshots).

### Шаг 3: Вставить флешку обратно в Xbox 360

Так как программа сохраняет обложки прямо в родную папку Aurora на флешке, **вам не нужно ничего никуда копировать**.
Просто извлеките флешку из ПК, вставьте в Xbox 360 и запустите Aurora — все обложки подтянутся автоматически!

## Особенности

- **Двуязычный интерфейс:** Полная поддержка русского и английского языков.
- **Интерактивность:** Удобное меню выбора скачиваемых ресурсов без ввода аргументов командной строки.
- **Безопасная отмена (Ctrl+C):** Если прервать работу программы нажатием `Ctrl+C`, она не закроется мгновенно (что могло бы привести к поломке файла), а аккуратно докачает текущую игру и выведет финальную статистику.

## Статусы скачивания

Во время работы программа отображает статусы для каждой игры:
- **`✓ загружено ресурсов: N`** (`✓ resources downloaded: N`) — программа успешно нашла и скачала новые ресурсы из интернета.
- **`пропущено (уже есть)`** (`skipped (already exists)`) — все запрошенные ресурсы уже присутствуют на вашей флешке. Программа пропустила игру мгновенно, не обращаясь к сети.
- **`✗ не найдено`** (`✗ not found`) — программа проверила Xbox Live и Archive.org, но в сети нет подходящих материалов.

## Структура файлов

Приложение создаёт точно такую же структуру файлов, как при FTP-загрузке через AuroraAssetEditor:

```
Data/GameData/
├── 415607E6_00000001/
│   └── GC415607E6.asset     ← обложка (boxart)
├── 4D5307E6_00000002/
│   └── GC4D5307E6.asset
└── ...
```
